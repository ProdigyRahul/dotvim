# Progress tracker

Check these off only when you can do them *without looking up keys*.

## Core grammar

- I think in "operator + motion/text object" (verb + noun).
- I use counts naturally (`3dw`, `2ci"`).
- I use `.` to repeat instead of redoing edits.

## Motions

- I use `f`/`t` and repeat with `;` / `,`.
- I use `%` to bounce between matching pairs.
- I use `{`/`}` and `(`/`)` to move by blocks/sentences.

## Text objects

- I use `ci"` / `ci)` / `ci}` / `dip` / `dap` daily.
- I use treesitter objects (`af`/`if`, `ac`/`ic`, `aa`/`ia`) without thinking.

## Search/replace

- I use `*` / `#` / `g*` correctly.
- I can do `cgn` + `.` renames safely.
- I can do `:%s/.../.../gc` with confirmations.
- I can use `:g` / `:v` to batch edit/delete lines.

## Registers/macros

- I use named registers (`"a`, `"b`) intentionally.
- I can paste a register in Insert mode (`<C-r>a`).
- I can record a macro (`qa ... q`) and replay (`@a`, `@@`) reliably.
- I can apply a macro to a visual selection using `:'<,'>normal @a`.

## Navigation memory

- I use marks (`ma`, `'a`, `` `a``).
- I use jumplist (`<C-o>`, `<C-i>`) to move through history.
- I use change list (`g;`, `g,`) to revisit edits.
